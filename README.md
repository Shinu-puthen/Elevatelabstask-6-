#  Task 6-Sales Trend Analysis

This task analyzes monthly sales data to understand revenue and order volume trends over time.

The process involved:
- Extracting the year and month parts from the order date.
- Calculating total monthly revenue using SQL's SUM function.
- Counting unique orders per month using COUNT(DISTINCT).
- Grouping the data by year and month.
- Sorting the results in chronological order.

This analysis helps reveal sales patterns, seasonal trends, and overall business performance by month, supporting data-driven decision-making.
